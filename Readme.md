# 🧩 AL Playground - Business Central

Bienvenido/a al **AL Playground**, un espacio donde experimento, pruebo y desarrollo funcionalidades en **AL para Microsoft Dynamics 365 Business Central**.  
Este repo no es una extensión final ni un producto — es más bien mi laboratorio de ideas 💡.

---

## 🚀 Objetivo

El propósito de este proyecto es:
- Probar nuevas funcionalidades del lenguaje **AL**.
- Explorar integraciones, patrones y buenas prácticas.
- Tener un entorno de test rápido para validar código antes de usarlo en proyectos reales.

---

## 🧠 Contenido del repo

Cada carpeta o archivo representa una prueba o experimento distinto.  
Por ejemplo:
- `CommonFx/`: pruebas en general, funciones reutilizables.
- `AIModels/`: pruebas de integracion con modelos de IA.
